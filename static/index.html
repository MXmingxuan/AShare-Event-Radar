📖 A股权威买方“事件雷达”

核心痛点与大模型局限性

在实际投资中，研究员每天面对大量的“混杂财务报表”、“日常股东会公告”。直接将几十篇长达万字的公告送给长窗口大模型，极易引发严重的大模型 **“注意力失焦 (Attention Dilution)”**，导致模型被废话带偏，忽略关键扭亏数字和重要动作。

本项目如何降噪破局：Map-Reduce 双模抽提

  • 第一层 [MAP] 粗挖提取：强制化利用大模型对单篇极长的财报进行“金矿提取”，压缩为只包含“资金数据”的小型 JSON 卡片。
  • 第二层 [SHUFFLE] Python 高效拦截：依托爬虫底表规则，我们可以在代码中精准销毁大量不具备实质增量贡献的“会议程序”、“减资进展”，并将具有同一**“事件暗号 (Event Key)”**的卡片拼凑聚合。
  • 第三层 [REDUCE] 高管总览研判：输入极度纯净、已被剔去几十万字杂质的【核心事件簇信息流】，强迫模型只做 S/A 级的宏观定调！绝少幻觉，高度可靠！
